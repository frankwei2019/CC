# Characters & World

## Main Characters (Character Anchors)

Use these descriptions verbatim in every Seedance prompt to maintain character consistency.

### 菠萝猫 (Pineapple Cat)
```
可爱的卡通菠萝猫角色，圆润的黄色身体上有菠萝纹理，大圆眼睛，
表情丰富夸张（开心/惊讶/大哭/得意），动作活泼夸张，
儿童绘本风格，中国古风卡通
```

### 大龙虾 (Big Lobster)
```
可爱的卡通大龙虾角色，红色外壳，住在海边贝壳房子里，
性格温和好奇，表情憨厚可爱，儿童绘本风格，中国古风卡通
```

## Locations

### 海边沙滩 (Beach)
```
阳光明媚的海边沙滩，金色沙子，蓝色大海，远处有礁石，
大龙虾的贝壳房子在海边，温暖舒适的氛围
```

### 大龙虾家 (Lobster's Home)
```
大龙虾的贝壳房子门口，海边风格，木门，周围有海星贝壳装饰，
阳光温暖
```

### 森林空地 (Forest Clearing)
```
森林中的空地，阳光透过树叶洒下斑驳光影，
周围有浆果丛和野花，宁静自然
```

### 古代厨房 (Ancient Kitchen)
```
中国古代乡村厨房，暖黄色火光，木制案板和厨具，
温馨质朴的农家氛围
```

### 花园 / 公园 (Garden / Park)
```
开满五颜六色鲜花的大花园，红色、黄色、紫色、粉色，
花海全景，花瓣随风飘落，美丽梦幻
```

## Forbidden Elements

Do NOT include these in children's content:
- 救护车、医院、急救场景
- 任何暴力或恐怖元素
- 说教式旁白（让角色对话传达道理）
- 复杂成人情绪（嫉妒、仇恨等）
